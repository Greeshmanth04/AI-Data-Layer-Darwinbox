# 🚀 Darwinbox AI Data Layer Engine

A production-grade full-stack Human Resources data platform with **fine-grained access control (RBAC)** and secure data governance.  
Ensures controlled data access at **collection, field, and row levels**.

---

## 🔥 Key Features

- 🔐 **Multi-Level Authorization**
  - Collection-level access  
  - Field-level restrictions  
  - Row-level filtering  

- 📊 **Data Visualization**
  - Dashboard  
  - Data Catalog  
  - Relationship Mapper  

- ⚙️ **Backend Architecture**
  - REST APIs with JWT Authentication  
  - Modular, scalable design  
  - Secure data access layer  

- 🗄️ **Database Optimization**
  - Efficient MongoDB schema  
  - Supports large-scale datasets  

---

## 🛠️ Tech Stack

- **Frontend:** React.js  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Auth:** JWT  
- **Tools:** Git, Postman
